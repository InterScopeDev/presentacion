Conceptos Básicos

    Wordpress

  • Sistema de gestión de contenidos (CMS).
  • Creación de webs empresariales, tiendas online, etc.
  • Sistema para publicar contenido en la web de forma sencilla.

    React

  • Librería open source de JavaScript para desarrollar interfaces de usuario.
  • Desarrollada por Facebook.
  • Contrucción de Interfaces de Usuario.

SLIDE 2

Ventajas y Desventas

    Wordpress

  • Es gratuito
  • Es de código abierto
  • Puedes instalar plugins muy útiles
  • Facilita el trabajo de SEO
  • No todo se puede hacer con WordPress
  • Personalización limitada
  • Es pesado
  • Tiene vulnerabilidades

    React

  • Aumenta la productividad y facilita un mayor mantenimiento
  • Asegura un renderizado más rápido (SPA)
  • Garantiza código estable
  • Es compatible con SEO
  • Existe React Native para el desarrollo de
    aplicaciones móviles (Producto Innovador)
  • Puedes Construir cualquier aplicación

    Estadisticas

    Wordpress

  • Según el sitio de WordCamp, actualmente hay más de
    75 millones de sitios que usan WordPress.
  • La herramienta representa el 59.7% de los sitios web creados
    por software y gana legítimamente el título de ” CMS más popular ”

    • React

  • Empresas y programadores en el mundo confian en el Ecosistema de REACT para desarrollar la siguiente generación de aplicaciones.
  • Algunos de estos ejemplos son: Facebook, Dropbox, Netflix, Airbnb, Paypal, Instagram, New York Times, Whatsapp, etc

    Proyecciones A corto plazo

  • Portafolio de Paginas
  • Administración del Galeria por parte del Cliente
  • Aplicaciones Móviles
  • Calendario de Citas y/o Eventos
  • Control de Estado de Resultados (Herramienta de Inventario)
